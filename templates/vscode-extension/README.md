# Dev Steps

1. Install recommended extensions.
2. run `pnpm i`.
3. run `npm run dev`.
4. Update `package.json`, your meta will be generated by `vscode-ext-gen` and `runonsave extension`.
5. Write your extension and debug it using F5.
